Option 1: Stack Vertically

How it works:

  • Title and navigation stack vertically on mobile (≤600px)
  • Both elements are centered
  • Title font size reduced slightly (22px)
  • Navigation links displayed as inline blocks with even spacing
  • Link font size reduced to 14px for better fit

Testing:

  • ✅ Resize your browser window to see the mobile layout
  • ✅ Or open DevTools (F12) and toggle device toolbar
  • ✅ Try widths: 320px (small phone), 375px (iPhone), 414px (large phone)

Pros: Clean, readable, large tap targets

Cons: Uses more vertical space
